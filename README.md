# dotfiles

## Installation

```bash
git clone https://github.com/aakashhemadri/dotfiles
cd dotfiles
source init.sh
```
## Usage

```bash
dots --help
dots use <profile>
dots install
```

## Configuration

```bash
vim config.yml
```
